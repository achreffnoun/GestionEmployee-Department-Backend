# Gestion Employee & Department

API gestion employee & department (SpringBoot)

## Employee 
 - GET: /employees
 - POST: /employees ( Test if employee already existing )
 - UPDATE: /employees/{id}
 - DELETE: /employees/{id}
## Department
 - GET: /departments
 - POST: /departments ( Test if department already existing )
 - UPDATE: /departments/{id}
 - DELETE: /employees/{id}
